## Requirements
- Serverless Framework already installed and configured
- NodeJS 22.x installed

## Deploy this demo

We will be using Serverless Framework and make sure you are running the latest version.

Deploy the project to the cloud:

```
serverless deploy
```

To delete the app:

```
serverless remove
```
